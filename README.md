# 건설 안전사고 데이터 분석 프로젝트

## 1. 프로젝트 개요

본 프로젝트는 **건설 안전사고 재해자 특성 데이터**를 활용하여 사고 발생 추이, 재해자 특성, 공사 관련 요인, 그리고 고위험 사고 패턴을 데이터 기반으로 종합 분석한 결과입니다. 이를 통해 건설 현장의 효율적인 안전 관리를 위한 핵심 인사이트를 도출하고자 했습니다.

## 2. 분석 환경 및 도구

- **언어**: Python
- **라이브러리**: Pandas, Matplotlib, Seaborn
- **데이터**: [국토안전관리원\_건설안전사고 재해자 특성](https://www.data.go.kr/data/15080650/fileData.do) (`data/raw/`)

## 3. 주요 분석 내용 및 결과

### 3.1. 사고 발생 추이 (Trend Analysis)

- **연도별/월별** 사고 발생 건수 집계 및 시각화.
- 전체적인 사고 발생 흐름과 특정 시기 집중 경향 파악.
- **결과물 경로**: `output/graphs/1_사고발생추이/`

### 3.2. 사고 유형 분석 (Accident Types)

- 빈번하게 발생하는 **상위 4대 사고 유형**과 그 외 유형 분석.
- 유형별 **평균 사망자 수**와 **부상자 수**를 비교하여 인명 피해 강도 확인.
- **결과물 경로**: `output/graphs/2_사고유형/`

### 3.3. 재해자 특성 분석 (Victim Characteristics)

- **재해자 특성 히트맵(Heatmap)**: 연령, 성별, 국적별 사망/부상 건수를 히트맵으로 시각화하여 취약 계층을 한눈에 파악.
- **특성별 비율 분석**: 성별, 국적, 연령대별 사망자와 부상자의 구성 비율(%)을 누적 막대 그래프로 비교 분석.
- **결과물 경로**: `output/graphs/3_재해자특성/`

### 3.4. 공사 관련 요인 분석 (Construction Factors)

- **공사 종류**: 건축, 토목 등 대분류별 사고 비율.
- **공종(소분류)**: 사고 다발 상위 10개 공종 및 '기타' 그룹 상세 분석.
- **공정율**: 공사 진행 단계(착공~완공)에 따른 사고 발생 빈도 분석.
- **결과물 경로**: `output/graphs/4_공사관련요인/`

### 3.5. 종합 위험 분석 (Comprehensive Risk Analysis)

- **공사 종류 × 사고 유형** 교차 분석을 통한 **Top 5 위험 패턴** 도출.
- 데이터 기반의 '최우선 관리 대상' 선정.
- **결과물 경로**: `output/graphs/5_종합위험분석/`

## 4. 프로젝트 폴더 구조

```bash
.
├── README.md               # 프로젝트 설명 및 보고서
├── data/
│   ├── raw/                # 원본 데이터 CSV
│   └── processed/          # 전처리 완료 데이터 CSV
├── notebooks/
│   └── analysis.ipynb      # 통합 분석 코드 (Jupyter Notebook)
└── output/
    └── graphs/             # 분석 결과 이미지(그래프) 저장소
        ├── 1_사고발생추이/
        ├── 2_사고유형/
        ├── 3_재해자특성/
        ├── 4_공사관련요인/
        └── 5_종합위험분석/
```

## 5. 실행 방법

1. `notebooks/analysis.ipynb` 파일을 엽니다.
2. 모든 셀을 순차적으로 실행하면 데이터 전처리부터 각 단계별 시각화 결과가 해당 폴더에 자동으로 저장됩니다.
